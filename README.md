CSE25 class project.



---

# Quick Start (Recommended)

Run these commands after cloning the repo.

### Windows PowerShell
git clone https://github.com/pabcal/CSE25-chess-eval.git
cd CSE25-chess-eval

python -m venv .venv
.venv\Scripts\Activate.ps1

pip install -r requirements.txt


### Mac / Linux
git clone https://github.com/pabcal/CSE25-chess-eval.git

python3 -m venv .venv
source .venv/bin/activate

pip install -r requirements.txt
